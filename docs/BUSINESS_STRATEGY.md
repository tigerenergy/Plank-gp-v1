## 🎯 비전 및 미션

### 비전
**"상급자와 직원이 서로 볼 수 밖에 없는 완전히 투명한 협업 플랫폼"**

### 미션
- **핵심 가치**: **상급자들도 놀 수가 없음** - 상급자와 직원이 서로의 작업을 볼 수 있어 누구도 게으름을 피울 수 없는 구조
- 기존 프로젝트 관리 도구의 계층적 권한 구조 문제 해결
- 한국 업무 문화에 맞춘 주간보고 기능 제공
- **양방향 투명성**: 상급자의 작업도 직원에게, 직원의 작업도 상급자에게 완전히 공개
- 실제 작업 활동을 자동으로 추적하여 보고서 작성 부담 최소화

## 📊 시장 분석

### 타겟 시장

#### 1. 한국 중소기업 및 스타트업 (1차 타겟)
- **시장 규모**: 한국 중소기업 약 600만 개
- **Pain Point**:
  - 엑셀 기반 주간보고의 비효율성
  - 관리자 권한 남용 문제
  - 데이터 손실 및 추적 불가능
- **Plank 솔루션**:
  - 자동화된 주간보고 (실제 작업 활동 자동 수집)
  - 관리자 없는 평등한 협업
  - 데이터 영구 보존

#### 2. 개발팀 및 IT 기업 (2차 타겟)
- **시장 규모**: 한국 IT 기업 약 5만 개
- **Pain Point**:
  - Trello는 개발 워크플로우에 최적화 안 됨
  - Jira는 복잡하고 비쌈
- **Plank 솔루션**:
  - Jira 스타일 시간 추적
  - 간단하고 직관적인 인터페이스
  - 한국어 완벽 지원

#### 3. 프리랜서 및 개인 프로젝트 (3차 타겟)
- **시장 규모**: 한국 프리랜서 약 50만 명
- **Pain Point**:
  - Trello 무료 버전 제한적
  - 주간보고 기능 없음
- **Plank 솔루션**:
  - 핵심 기능 무료 제공
  - 개인 프로젝트 관리 최적화

## 🚀 핵심 차별화 포인트

### 1. 완전한 투명성: 상급자와 직원이 서로 볼 수 밖에 없는 구조 ⭐ **최대 차별화**

**핵심 가치**: **상급자들도 놀 수가 없음** - 이것이 Plank의 핵심 포인트입니다.

- **Trello/Asana의 한계**: 
  - 계층적 권한 구조
  - 관리자는 직원의 작업만 볼 수 있지만, 자신의 작업은 숨길 수 있음
  - 직원은 관리자의 작업을 제대로 볼 수 없어 불평등감 발생
- **Plank의 해결책**: 
  - **상급자(관리자)와 직원이 서로 볼 수 밖에 없는 구조**
  - 상급자의 작업도 직원들에게 완전히 공개됨
  - 직원의 작업도 상급자에게 완전히 공개됨
  - **상급자들도 놀 수 없음** - 자신의 작업이 직원들에게 보이므로 책임감이 생기고, 실제로 일을 해야 함
  - 모든 멤버 동등한 권한, 작성자만 수정 가능
- **시장 가치**: 
  - 투명성과 공정성에 대한 수요 증가
  - 상급자와 직원 간의 신뢰 구축
  - 양방향 책임감으로 생산성 향상

### 2. 통합 주간보고 기능 ⭐ **Trello에 없는 기능**
- **기존 방식의 문제**: 엑셀 수동 입력, 데이터 손실, 관리자 복붙
- **Plank의 해결책**:
  - 실제 작업 활동 자동 수집 (카드 생성/수정/삭제/이동, 체크리스트, 댓글)
  - Jira 스타일 시간 추적 및 자동 집계
  - 웹페이지로 실시간 공유
  - 데이터 영구 보존
- **시장 가치**: 한국 기업의 주간보고 문화에 완벽히 맞춤

### 3. 실시간 협업 (피그마/엑셀 공유 시트 스타일)
- **기존 방식의 문제**: 엑셀 동시 편집 충돌, 버전 관리 없음
- **Plank의 해결책**: Supabase Realtime으로 실시간 자동 업데이트
- **시장 가치**: 현대적인 협업 경험 제공

### 4. 데이터 영구 보존
- **기존 방식의 문제**: 매주 데이터 삭제, 통계 분석 불가
- **Plank의 해결책**: 모든 주간보고 데이터 영구 보관, 통계 분석 가능
- **시장 가치**: 장기적인 성과 추적 및 분석

  - 비쌈 ($10.99/월)
  - 주간보고 기능 제한적
- **Plank 차별화**: 간단하고 직관적, 주간보고 특화

- **강점**: 개발팀에 최적화, 강력한 기능
- **약점**: 
  - 너무 복잡함
  - 비쌈 ($7.75/월)
  - 스타트업에는 과함
- **Plank 차별화**: 간단하면서도 Jira 스타일 시간 추적

- **강점**: 올인원 워크스페이스
- **약점**: 
  - 학습 곡선 높음
  - 주간보고 기능 없음
  - 느린 성능
- **Plank 차별화**: 주간보고 특화, 빠른 성능

### 개인 사용자 (직원)
**"실제 작업한 내용이 자동으로 주간보고에 반영되고, 상급자의 작업도 볼 수 있습니다"**
- 수동 입력 최소화
- 시간 절약
- 정확한 보고
- **상급자의 작업도 볼 수 있어서 "상급자도 열심히 일하고 있구나" 확인 가능**

### 팀/기업
**"상급자와 직원이 서로 볼 수 밖에 없는 완전히 투명한 협업 환경"**
- **양방향 투명성**: 상급자의 작업도 직원에게, 직원의 작업도 상급자에게 완전히 공개
- **상급자들도 놀 수 없음**: 자신의 작업이 직원들에게 보이므로 책임감이 생김
- 데이터 영구 보존
- 실시간 협업

### 관리자/리더 (상급자)
**"팀원들의 작업 현황을 실시간으로 확인하고, 내 작업도 팀원들에게 보여서 책임감을 갖습니다"**
- 실시간 대시보드
- 통계 및 분석
- 데이터 기반 의사결정
- **내 작업도 직원들에게 보여서 "상급자도 열심히 일하고 있구나" 신뢰 구축**
- **상급자도 놀 수 없음**: 자신의 작업이 직원들에게 보이므로 실제로 일을 해야 함

## 🎨 브랜딩 전략

### 브랜드 메시지
**"상급자와 직원이 서로 볼 수 밖에 없는 완전히 투명한 협업 플랫폼"**

**서브 메시지**: **"상급자들도 놀 수 없음"** - 이것이 Plank의 핵심 포인트입니다.

### 핵심 키워드
- 양방향 투명성 (Bidirectional Transparency)
- 상급자도 놀 수 없음 (No Hiding for Anyone)
- 완전한 공개 (Full Visibility)
- 자동화 (Automation)
- 한국 특화 (Korea-First)

### 타겟 페르소나
1. **개발자/프로덕트 매니저 (직원)**: 
   - 간단하고 효율적인 도구 원함
   - 상급자의 작업도 볼 수 있어서 "상급자도 열심히 일하고 있구나" 확인 가능
2. **팀 리더 (상급자)**: 
   - 팀원들의 작업을 투명하게 추적하고 싶음
   - **내 작업도 팀원들에게 보여서 책임감을 갖고 싶음**
   - **상급자도 놀 수 없음**: 자신의 작업이 직원들에게 보이므로 실제로 일을 해야 함
3. **중소기업 대표**: 
   - 비용 효율적이고 실용적인 도구 원함
   - 상급자와 직원 간의 신뢰 구축을 위한 투명성 원함
