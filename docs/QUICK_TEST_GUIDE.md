## 🎯 빠른 테스트 순서 (5분)

### 1단계: 시간 추적 기능 (1분)
1. 보드 접속 → 카드 열기
2. "시간 추적" 탭 클릭
3. 시간 로그 생성 (2시간, 오늘 날짜)
4. 저장 버튼 클릭 → 중복 클릭 방지 확인 (스피너 표시)
5. 생성된 로그 확인

### 2단계: 주간보고 작성 (2분)
1. "주간보고 작성" 버튼 클릭
2. 자동 수집 확인:
   - ✅ 완료된 카드 자동 포함
   - ✅ 진행 중인 카드 자동 포함
   - ✅ 시간 자동 집계
3. 진행 중인 작업 정보 입력:
   - 상태: "진행중"
   - 진척도: 50%
   - 작업 시간: 8시간
   - 설명: "테스트 중"
4. "임시 저장" 클릭 → 중복 클릭 방지 확인
5. "제출" 클릭 → 중복 클릭 방지 확인

### 3단계: 공유 페이지 확인 (1분)
1. 자동으로 공유 페이지 이동 확인
2. 사용자별 보고서 카드 표시 확인
3. "통계" 버튼 클릭 → 통계 페이지 확인
4. "내보내기" 버튼 클릭 → PDF/CSV 다운로드 확인
5. "수정 이력" 버튼 클릭 → 이력 모달 확인

### 4단계: 실시간 업데이트 (1분)
1. 새 탭에서 같은 공유 페이지 열기
2. 원래 탭에서 주간보고 수정
3. 새 탭에서 자동 업데이트 확인 (새로고침 없이)
